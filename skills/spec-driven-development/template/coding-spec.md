| Layer | Directory | Description |
|:------|:----------|:------------|
| Transport | {dir_transport} | HTTP/gRPC handlers |

These constraints CANNOT be modified. See [_shared/domain-architecture-reference.md](../../_shared/domain-architecture-reference.md):
